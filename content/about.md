## このブログについて

「ググっても分からなかった初心者へ」は、プログラミングや技術でつまずいた初心者のための解決ノートです。

検索しても解決しなかったあの問題を、一緒に解決します。

## 運営者について

プログラミングを独学で学び始めた頃、何度も「ググっても分からない」壁にぶつかりました。エラーメッセージをそのまま検索しても、出てくるのは英語の技術文書や上級者向けの解説ばかり。「初心者の自分が本当に知りたいことが書いてあるサイトがない」――そんな経験を何度も繰り返す中で、同じ悩みを持つ人のために自分が発信しようと決意しました。

今もまだ学び続けている途中です。だからこそ、初心者がどこでつまずくのか、何が分からないのかが分かります。「完璧な知識を持つ先生」ではなく、「少し先を歩く仲間」として、一緒に成長していくスタンスで記事を書いています。

## 実績

- 📝 ブログ記事20本以上を執筆・公開
- 🐍 Python・Git・HTML・CSSなど、初心者向け解説を幅広く執筆
- 📘 Kindle：「ChatGPTの教科書」「ChatGPT × Excel 完全攻略」
- 🤖 姉妹ブログ「AIつかいこなし帳」を運営

## 発信しているテーマ

- 🐍 Python の基礎とエラー解決
- 🌿 Git / GitHub の使い方
- 🎨 HTML / CSS のつまずきポイント
- ⚡ VS Code や開発環境の設定
- 🤖 AIツールを使ったプログラミング

## 大切にしていること

- 初心者が本当につまずくポイントを丁寧に解説する
- 専門用語を使ったら必ず一言で説明を添える
- 「簡単です」とは言わない。一緒に解決する姿勢で書く
- 経験者が読んでもくどくない、シンプルな構成
- 分からないことは正直に「分からない」と書く。知ったかぶりはしない
