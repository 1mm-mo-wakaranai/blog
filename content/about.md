## このブログについて

「ググっても分からなかった初心者へ」は、プログラミングや技術でつまずいた初心者のための解決ノートです。

検索しても解決しなかったあの問題を、一緒に解決します。

## 運営者について

プログラミングを独学で始めたけど、エラーが出るたびにパニックになるタイプ。ググっても英語の技術文書ばかりで「日本語で、初心者の私にも分かるように書いてくれ…」って何度も思いました。

今もまだ全然分かってないことだらけだけど、全知全能くん（AI相棒）に助けてもらいながら少しずつ前に進んでます。「完璧な先生」じゃなくて「一緒につまずいてる仲間」として記事を書いてます。

## 実績

- 📝 ブログ記事20本以上を執筆・公開
- 🐍 Python・Git・HTML・CSSなど、初心者向け解説を幅広く執筆
- 📘 Kindle：「ChatGPTの教科書」「ChatGPT × Excel 完全攻略」
- 🤖 姉妹ブログ「AIつかいこなし帳」を運営

## 発信しているテーマ

- 🐍 Python の基礎とエラー解決
- 🌿 Git / GitHub の使い方
- 🎨 HTML / CSS のつまずきポイント
- ⚡ VS Code や開発環境の設定
- 🤖 AIツールを使ったプログラミング

## 大切にしていること

- 初心者が本当につまずくポイントを丁寧に解説する
- 専門用語を使ったら必ず一言で説明を添える
- 「簡単です」とは言わない。一緒に解決する姿勢で書く
- 経験者が読んでもくどくない、シンプルな構成
- 分からないことは正直に「分からない」と書く。知ったかぶりはしない
